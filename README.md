## Installation & Setup

### Docker
To set up the project using Docker, follow these steps:
1. Ensure you have Docker installed on your machine.
2. Clone the repository using the command:
   ```
   git clone https://github.com/hr628/Online-Examination-Management-System.git
   ```
3. Navigate to the cloned directory:
   ```
   cd Online-Examination-Management-System
   ```
4. Build the Docker image:
   ```
   docker build -t online-exam-management-system .
   ```
5. Run the Docker container:
   ```
   docker run -p 8080:80 online-exam-management-system
   ```
6. Access the application at `http://localhost:8080`.
